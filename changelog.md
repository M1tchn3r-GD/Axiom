# Axiom Changelog

## v1.0.0 - Initial Release

### 🎉 Features
- **Noclip**: Pass through all obstacles without collision
  - Prevents player death
  - Anti-cheat bypass for jump counters
  
- **Speed Hack**: Adjustable game speed
  - Range: 0.0001x to 100x
  - Default: 1.0x (normal speed)
  - Smooth speed transitions
  
- **Instant Complete**: Auto-complete levels
  - Triggers completion animation after 0.1s
  - Works with all level types
  
- **FPS Bypass**: Unlock frame rate
  - Configurable target FPS (1 - 520)
  - Default: 240 FPS
  - Maintains physics accuracy through frame subdivision
  
- **Practice Music Fix**: Sync music in practice mode
  - Continuous audio synchronization
  - Fixes music desync issues
  - Improves practice mode experience

### 🎨 UI/UX
- Custom Axiom button in main menu
- Quick access to mod settings
- Clean settings interface with descriptions

### 🔧 Technical
- Built with Geode SDK v4.0.0
- Compatible with GD 2.2081 (Win/Mac/iOS/Android)
- C++23 codebase
- Modular hook architecture

### 📚 Documentation
- Comprehensive README
- Feature usage guide
- Build instructions
- Development guidelines

### 🛡️ Safety
- Anti-cheat bypass measures
- Smooth hook implementations
- No game crashes or instability
