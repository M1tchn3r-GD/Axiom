# About Axiom

## 🎯 What is Axiom?

Axiom is a comprehensive hack client for Geometry Dash, built from the ground up using the Geode SDK and developed entirely in VSCode. It provides powerful gameplay modifications and quality-of-life features for both casual players and level creators.

## 🎨 Philosophy

Axiom is designed with three core principles:

1. **Accessibility** - Easy to use for beginners, powerful for advanced users
2. **Stability** - Robust implementations that don't crash your game
3. **Modularity** - Each feature works independently and can be toggled individually

## 🔬 Technical Details

### Architecture
Axiom uses Geode's powerful hooking system to modify game behavior at runtime. Each feature is implemented as a separate hook class, ensuring clean separation of concerns and minimal interference between features.

### Key Technologies
- **Geode SDK v4.0.0** - Modern modding framework
- **C++23** - Latest C++ standard for performance and safety
- **CMake** - Cross-platform build system
- **Node IDs** - For reliable UI element access

### Platform Support
- ✅ Windows (2.2081)
- ✅ macOS (2.2081) - Both Intel and Apple Silicon
- ✅ iOS (2.2081)
- ✅ Android (2.2081)

## 🎮 Use Cases

### For Players
- **Practice Mode**: Use noclip to explore levels without dying

## 🔐 Safety & Ethics

### What Axiom Does
- Modifies local gameplay experience
- Works entirely client-side
- Doesn't modify game files permanently

### What Axiom Doesn't Do
- ❌ Doesn't hack online servers
- ❌ Doesn't steal accounts or data
- ❌ Doesn't modify other players' games
- ❌ Doesn't affect online leaderboards directly

### Ethical Use
We recommend using Axiom for:
- Personal practice and improvement
- Level creation and testing
- Educational purposes
- Entertainment on offline levels

Please be respectful of the game and community. Avoid using hacks to unfairly claim records or deceive others.
